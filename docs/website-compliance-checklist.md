# La Maison Homes — Website Compliance Checklist

## A. Entity and transparency
- [ ] Correct legal entity name displayed
- [ ] Contact details are current
- [ ] ABN/business details displayed where needed
- [ ] No misleading location or service claims
- [ ] No unsupported "guaranteed" wording unless legally backed

## B. Public legal pages
- [ ] Terms of Use published
- [ ] Privacy Policy published
- [ ] Cookie Policy published if relevant
- [ ] Direct Booking Terms published
- [ ] Cancellation / Refund Policy published
- [ ] House Rules or guest standards linked where relevant
- [ ] Payment terms visible before checkout

## C. Booking flow
- [ ] User can access terms before paying
- [ ] Checkbox or equivalent assent mechanism exists
- [ ] Cancellation terms visible before booking confirmation
- [ ] Total price and fees clearly shown
- [ ] Bond/deposit rules disclosed
- [ ] ID verification requirement disclosed if used
- [ ] House rule obligations disclosed
- [ ] Occupancy limits disclosed
- [ ] Check-in and check-out rules disclosed

## D. Consumer law / representations

Check website claims for:
- [ ] "professionally managed"
- [ ] "verified"
- [ ] "premium"
- [ ] "luxury"
- [ ] "secure booking"
- [ ] "best rates"
- [ ] "guest support"
- [ ] "hotel-grade cleaning"
- [ ] "instant confirmation"

For each claim, confirm:
1. [ ] Operational basis exists
2. [ ] Wording is not misleading
3. [ ] Supporting internal SOP exists where needed

## E. Privacy and data
- [ ] Data collection points identified
- [ ] Contact forms include consent wording
- [ ] Marketing opt-in wording separated where needed
- [ ] Data retention logic documented
- [ ] Third-party providers listed or covered in privacy wording
- [ ] Payment processing handled by compliant provider
- [ ] Sensitive personal data use disclosed where required
- [ ] ID document handling process documented
- [ ] Cookie/tracking tools disclosed if in use

## F. Payments
- [ ] Payment method disclosure accurate
- [ ] Refund timing stated
- [ ] Chargeback / disputed transaction handling internally documented
- [ ] Security deposit / bond terms clear
- [ ] Extra charges policy clear
- [ ] Damage recovery process supported by terms

## G. Third-party systems
- [ ] OTA sync limitations internally documented
- [ ] Hostaway or PMS sync risk known
- [ ] Message sync risk known
- [ ] Rate and availability disclaimer considered
- [ ] Platform mismatch escalation workflow documented

## H. Operational promise alignment
- [ ] Housekeeping wording matches actual cleaning SOP
- [ ] Support wording matches actual support hours and process
- [ ] Property standards wording matches inspection process
- [ ] Booking confirmation wording matches system capability
- [ ] Amenities wording matches listing verification process

## I. Complaints and disputes
- [ ] Contact path for complaints available
- [ ] Response workflow documented internally
- [ ] Refund decision matrix exists
- [ ] Incident evidence process exists
- [ ] Escalation rules for serious matters exist

## J. Maintenance
- [ ] Website legal pages version controlled
- [ ] Review date set
- [ ] Legal register updated after each revision
